# le3corde

Website of 3 cordes

## GitHub Pages Setup

This is a static website hosted on GitHub Pages.

### File Structure

```
/
├── index.html          # Main landing page
├── styles/
│   └── style.css       # Global stylesheet
└── README.md           # This file
```

### Customization

1. **Update the title and branding:**
   - Edit the `<title>` tag in `index.html`
   - Change "MyProject" to your project name
   - Modify colors in `styles/style.css` by changing CSS variables in `:root`

2. **Add more pages:**
   - Create new `.html` files in the root directory
   - Link to them in the navbar

3. **Deploy:**
   - Push changes to your GitHub repository
   - Go to Settings → Pages and select "Deploy from a branch"
   - Choose the `main` branch (or your default branch)
   - Your site will be live at `https://yourusername.github.io/le3corde/`

### Quick Start

- Edit `index.html` to add your content
- Modify colors in `styles/style.css`
- Commit and push to GitHub
